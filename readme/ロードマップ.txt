１．Ver0.02まで
	最低限「キャラと遊んで仲良くなり、恋慕などの関係性にバグなく発展出来る」程度まで機能の仮組を終わらせます。
	よって大きな変更点は
		・各種調教コマンドの整備と追加
		・恋慕以外の関係性実装、及び関係性の更なる発展実装
		・弱点コマンドなど調教関連のシステム実装
		・上記全てに伴うTALENT・ABL・JUEL・PALAMの調整・追加
	が主となります。

	ここまで終われば最低限キャラを追加できるようになるはず。

２．Ver0.03まで
	戦闘システムとリゾートの経営システム、ZPの使用先拡充を実装予定です。

	・戦闘システムについて
	探索を「簡易探索」と「特別探索」に分け、後者で探索メンバーを使用した戦闘が出来るようにします。
	また探索先を複数実装し、得られるルピや素材、出現する敵の強さなどで差別化します。

	・リゾート経営システムについて
	詳細は未定ですが、現在少しだけ実装している雑務ゲージや従業員変数と関わりのあるものにします
	また上の戦闘システムと連動し、ボスを倒した場合は新たなリゾートマップが解放されるようにします。

	・ZPの使用先拡充について
	１．リゾート地に来られないような設定のキャラをフォロー
		境界の世界に居たり、既に死んでいたり、明確に敵対していたりするキャラを実装します
		EXキャラとは別扱いになる予定
		素材を支払って時を超えたり、キャラ本人ではない写身を作成したりと複数のアプローチを行う予定です
	２．催眠や触手、セックスしないと出られない部屋など非現実的なプレイをフォロー
		プレイの幅を広げるために基本的になんでも出来るようにします
		実装するプレイは作者の趣味が大いに反映されます

	ここまで来れば一端の「遊べるバリアント」になっているはず。
