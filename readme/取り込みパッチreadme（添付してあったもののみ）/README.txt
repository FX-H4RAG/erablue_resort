カットインアニメーションをするアニメスプライトを呼び出す仕組みです。

カットインシステム素材
・カットイン呼出.ERB
・カットイン用.ERH
・チェインバーストカットイン.ERB
・奥義カットイン.ERB


追加する必要のある要素
・CALL カットイン呼出();奥義、チェインバースト
・CALL カットイン終了
・アニメ選択オプション　キャラごとに自分が出したいアニメを指定する変数

読み込む画像のフォーマット
サイズは不問で1:2の比率のもの。優先度はオプションで指定したもの＞顔_デフォルト＞カットイン出さないです。
・縦カットイン.png
・横カットイン.png

"CALL カットイン呼出"でアニメを呼び出して、"CALL カットイン終了"でアニメを停止、削除します。
停止、削除しない限り画面に残り続けるので、必要なだけアニメーションさせた後、終了させます。
私は戦闘終了時（倒した時）と次行動選択時（倒せなかったとき）にしていましたが、例えば倒せない場合で、
敵が多数行動してきた場合、アニメがループ再生される可能性が高くなります。
また、AWAITなどのEMUERAの描画を強制停止させるようなコマンドはアニメーションも止めてしまいます。
